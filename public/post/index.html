🎉 mongodb exporter 下载： wget https://github.com/percona/mongodb_exporter/releases/download/v0.11.2/mongodb_exporter-0.11.2.linux-amd64.tar.gz 减压后将 mongodb_exporter 二进制文件移动： mv mongodb_exporter /usr/local/bin/ 在 MongoDB 中创建一个名为 prometheus 的用户 进入mongodb mongo mongo.skydns.local --ssl --sslPEMKeyFile /etc/ssl/mongo/client.pem --sslCAFile /etc/ssl/mongo/ca.crt --authenticationDatabase '$external' --authenticationMechanism MONGODB-X509 创建用户 use admin db.createUser({ user: "prometheus", pwd: "prometheus", roles: [ { role: "read", db: "admin" }, { role: "readAnyDatabase", db: "admin" }, { role: "clusterMonitor", db: "admin" } ] }); 创建service文件 ‼️ 修改证书路径 和 IP cat <<EOF > /etc/systemd/system/mongodb_exporter.service [Unit] Description=mongodb_exporter After=network.
